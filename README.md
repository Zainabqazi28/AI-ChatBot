# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview

This project is a **Rule-Based AI Chatbot** developed as **Project 1** for the DecodeLabs Artificial Intelligence Internship. The chatbot interacts with users by responding to predefined messages using simple **if-else** decision-making logic.

The application features a user-friendly web interface built with **HTML**, **Bootstrap**, and **JavaScript**, while the chatbot logic is implemented in **Python** using the **Flask** framework.

---

## 🚀 Features

* Interactive chatbot interface
* Handles greetings (Hi, Hello)
* Responds to common questions
* Handles exit commands (Bye, Exit)
* Uses rule-based AI (if-else conditions)
* Responsive Bootstrap UI
* Simple and beginner-friendly implementation

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Rule-Based-AI-Chatbot.git
```

2. Navigate to the project folder

```bash
cd Rule-Based-AI-Chatbot
```

3. Install dependencies

```bash
pip install flask
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```

## 🧠 Chatbot Logic

The chatbot uses predefined rules implemented through **if-else** statements. It compares the user's input with known commands and returns the appropriate response. If no matching rule is found, it displays a default message indicating that it does not understand the input.
